1. 将网易云音乐nodejs api的项目在本地跑起来。 

相关文档地址
https://binaryify.github.io/NeteaseCloudMusicApi/#/

clone项目到本地，运行如下指令即可。

```
> node app.js
```

2. 新增如下代码到本地hosts文件，给127.0.0.1指定一个域名

```js
127.0.0.1 devlocal.feewee.cn
```

2. 将本项目clone到本地，并运行如下指令跑起来

```js
> yarn start
```

最后使用`devlocal.feewee.cn:3001`访问本项目。
